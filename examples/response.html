Semi-log Plot

Plot Semi-log graphs. This example plots a graph of a Low-pass filter frequency response. See the code below:

Example code

var graph = new Graph({
    appendTo : "container",
    canvasWidth : 800,
    canvasHeight : 800/Math.sqrt(2),
    title : "Low-pass Filter Frequency Response",
    xAxisTitle : "Frequency (Hz)",
    yAxisTitle : "Attenuation (dB)",
    xAxisScale : "log",
    xAxisNumSteps : 5,
    lineShowPoints : true				
});

var py = [-0.5, -0.8, -0.4, -0.4, -0.4, -0.7, -0.7, -0.3, -0.4, -0.3, 
    -0.2, -0.8, -6.5, -16.8, -27.3, -36.7, -44.8, -55.5, -47.9, -45.6],

var px = [1, 2, 3, 6, 11, 21, 38, 70, 127, 234, 428, 785, 1438, 2637, 4833, 
    8859, 16238, 29764, 54556, 100000];
				
graph.plot(py, px, {} /* autoscale */, [-60, 0]/* dB */);
